Divider

A standard divider

If you are looking to add a divider in between grid rows, you can use the ui divided grid variation. For more information check out the grid definition

Vertical Divider

A divider can segment content vertically

Vertical and horizontal dividers require you to add position: relative to the element that you would like to contain the divider
A vertical divider will automatically adjust to a horizontal divider at mobile resolutions when used inside a stackable grid

Horizontal Divider

A divider can segment content horizontally

Horizontal dividers can also be used in combination with headers and icons to create different styles of dividers.
Dividers will automatically vary the size of their dividing rules to match the length of your text
